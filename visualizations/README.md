# Visualizations

Interactive HTML visualizations generated from the GoCarbonTracker knowledge base. Open any `.html` file in a browser to explore.

| File | Description |

| `vyuh_tier_country_all_companies.html` | 216 companies mapped by supply chain tier and country |
| `vyuh_tier_separate_all_companies.html` | Separate tier view of all companies |
| `vyuh_format_complete.html` | Complete format visualization |
| `vyuh_format_with_relationships.html` | Company relationships network |
| `vyuh_format_scope_emissions.html` | Scope emissions visualization |
| `vyuh_country_hq_kb_companies.html` | Companies by country of headquarters |
| `energy_initiatives_animated.html` | Animated energy initiatives network |
